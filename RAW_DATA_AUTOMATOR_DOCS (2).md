## 4. 매체별 처리 스펙

### 4-1. Youtube

| 항목 | 내용 |
|------|------|
| 파일 형식 | CSV |
| 인코딩 | **UTF-16** |
| 구분자 | 탭(`\t`) |
| 헤더 위치 | 3행 (0-index: 2) |
| 상품명(B열) | 캠페인명에서 자동 감지 (`VRC`, `Bumper`, `FPM`, `FPM_Shorts`) |
| 비용 수식 | 원본 그대로 (`net` 컬럼) |
| 비용 컬럼명 | `net` |
| 날짜 컬럼명 | `일` |
| 광고명 컬럼명 | `광고 이름` |
| 노출 컬럼명 | `노출수` |
| 클릭 컬럼명 | `클릭수` |
| 조회 컬럼명 | `TrueView 조회수` |
| 동영상 25~100% | `동영상 25% 재생` ~ `동영상 100% 재생` |
| BA Q1 필터링 | 불필요 (파일 자체가 BA Q1) |
| 처리 결과 | 36행 (테스트 기준) |

**컬럼 매핑 (구글 애즈 리포트 → RAW Data)**
```
캠페인      ← 캠페인
그룹        ← 광고그룹
광고        ← 광고 이름
일자        ← 일
비용(net)   ← net
노출        ← 노출수
클릭        ← 클릭수
조회        ← TrueView 조회수
25~100      ← 동영상 25~100% 재생
```

### 4-2. Naver GFA

| 항목 | 내용 |
|------|------|
| 파일 형식 | CSV |
| 인코딩 | **UTF-8-SIG** |

| 노출 컬럼명 | `노출` |
| 클릭 컬럼명 | `클릭` |
| BA Q1 필터링 | **필요** (`캠페인 이름`에 `BA Q1` 포함 여부) |
| 처리 결과 | 27행 (BA Q1 필터링 후) |

**상품명(B열) 자동 판단 규칙**
```
캠페인명에 'Feed'  포함 → Feed Carousel
캠페인명에 'DMP'   포함 → M.Branding DMP
그 외               → M.Branding
```

**컬럼 매핑**
```
캠페인      ← 캠페인 이름
그룹        ← 광고 그룹 이름
광고        ← 광고 소재 이름
일자        ← 기간 (형식 변환)
비용(net)   ← 총 비용 / 1.1 * 0.95
노출        ← 노출
클릭        ← 클릭
```

## 8. 텍소노미 기반 자동완성 로직 (신규)

### 개요
매체 세팅 시 텍소노미 코드를 캠페인명/소재명에 심어두면, 리포트에서 해당 코드를 역매핑하여 A, B, F, G열을 자동완성하는 방식.
기존 키워드 추정 방식보다 정확하며, `맥도날드_자동화_작업용.xlsx`의 `DATA RAW` 시트를 코드 사전으로 활용.

### 코드 사전 구조 (`DATA RAW` 시트)

| 섹션 | 열 | 내용 |
|------|-----|------|
| 매체 코드표 | 2열(MEDIA), 4열(CODE), 6열(최종CODE), 8열(Alias) | 최종CODE → 매체명 |
| 상품 코드표 | 9열(PRODUCT), 11열(CODE), 13열(최종CODE) | 최종CODE → 상품명 |
| 소재 코드표 | 24열(소재), 25열(가로세로), 26열(초수) | 소재명/방향/초수 |

- 매체 코드: 63개 등록
- 상품 코드: 165개 등록

### 캠페인명 파싱 → A열(미디어), B열(상품명)

```
2602_KAKA1_BIZB0_BA_Q1
  [0]  [1]   [2]  [3][4]
  날짜  매체코드 상품코드 캠페인

KAKA1 → DATA RAW 매체표 조회 → Kakao Moment  (A열)
BIZB0 → DATA RAW 상품표 조회 → Bizboard      (B열)
```

### 소재명 파싱 → F열(소재), G열(초수)

```
M_Snackwrap_H_15_A
[0]  [1]      [2] [3] [4]
구분  소재명   방향  초수  매개변수

Snackwrap → 스낵랩  (F열)
15        → 15      (G열)
```

### 매체별 텍소노미 위치

| 매체 | 캠페인코드 위치 | 소재코드 위치 | 비고 |
|------|--------------|-------------|------|
| Kakao | `캠페인 이름` 컬럼 | `소재 이름` 컬럼 | |
| Naver GFA | `캠페인 이름` 컬럼 | `광고 소재 이름` 컬럼 | |
| Youtube | `캠페인` 컬럼 = 소재코드 | `캠페인` 컬럼 | 매체/상품은 `광고 이름` 앞부분으로 판단 |

### 소재명 영→한 변환 테이블

| 영문 | 한글 |
|------|------|
